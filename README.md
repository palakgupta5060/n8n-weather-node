# 🌦️ n8n Weather Node

A custom n8n community node built with TypeScript that fetches live weather information using the Open-Meteo API.

## 🚀 Features

- Search weather by city
- Automatic geocoding (city → coordinates)
- Live temperature
- Humidity
- Wind speed
- Country & location details
- Built as a custom n8n Community Node

## 🛠 Tech Stack

- TypeScript
- n8n
- Open-Meteo Geocoding API
- Open-Meteo Weather API
